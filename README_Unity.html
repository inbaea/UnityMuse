Welcome to Unity example app for Libmuse.

This example app is a getting started point for developing apps for the muse headband in Unity. It uses the Libmuse library for interfacing with Muse headbands, including finding paired Muses, connecting to them, reading their state, and receiving packets for raw EEG data and other values.

You can use the example application as a starting point for developing your own Unity applications.

Documentation

This example code requires you copy the Libmuse library files for the platform you wish to target.

  • iOS - copy Muse.framework from the iOS SDK archive into Assets/Plugins/iOS
  • Android - copy libs from the Android SDK archive into Assets/Plugins/Android
  • Windows - copy libmuse.dll from the Windows SDK archive into Assets/Windows
Open the project in Unity Hub and choose your installed version to apply to the project.

After opening the project in Unity, load the scene file libmuseScene.unity.
Select Canvas and open the the Inspector tab.

Attach the SampleApp.cs script to the canvas then attach all the canvas components to the script.

Configure your build and player settings as appropriate for your platform.
